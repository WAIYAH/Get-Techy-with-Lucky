Introduction to HTML

This page introduces the basics of HTML, including headings, forms, and layout.

Headings

HTML provides six levels of headings, from <h1> to <h6>, with <h1> being the highest level.

Forms

Forms are used to collect user input. Common form elements include text fields, checkboxes, radio buttons, and submit buttons.

Layout

HTML layout can be managed using various elements such as <div>, <span>, and CSS for styling and positioning.
